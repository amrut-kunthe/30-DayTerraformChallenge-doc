# 30-DayTerraformChallenge-doc

Documenting my 30-day Terraform challenge, focusing on Infrastructure as Code (IaC) fundamentals, AWS-based hands-on labs, and real-world use cases. This repository captures daily progress, code, and key learnings to build scalable, automated, and secure cloud infrastructure.

This repository documents my participation in the 30 Days of Terraform Challenge, organized by the AWS AI/ML User Group Kenya, Meru HashiCorp User Group, and EveOps communities.
The challenge is designed to build strong hands-on expertise in Terraform and Infrastructure as Code (IaC) through daily practical exercises and real-world cloud scenarios.
Throughout these 30 days, I will be working on progressively advanced topics—from Terraform fundamentals to building scalable, automated, and secure cloud infrastructure. This repository captures my daily progress, code implementations, key learnings, and challenges faced along the way.

**Key Objectives**
- Develop practical Terraform skills through hands-on labs
- Understand core IaC concepts and best practices
- Build and manage cloud infrastructure efficiently
- Learn modular, reusable, and scalable Terraform design
- Strengthen real-world cloud automation capabilities

**What to Expect**
- Daily updates (Day 1–Day 30)
- Terraform code and configurations
- Real-world use cases and mini projects
- Notes, tips, and troubleshooting insights
